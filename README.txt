# ⛏️ El Yacimiento Olvidado - Análisis en Power BI

## 📊 Descripción
Dashboard interactivo en Power BI que integra y analiza datos arqueológicos.  
Incluye información de registros de campo y análisis de laboratorio para explorar la historia cultural detrás de los hallazgos, permitiendo identificar patrones históricos y tendencias en los datos.

## 🎯 Objetivos
- Integrar datos de distintas fuentes arqueológicas (registros de campo y laboratorio).  
- Limpiar y transformar los datos para análisis consistente.  
- Construir un modelo de datos eficiente en estrella, incluyendo tabla calendario.  
- Analizar tipos de artefactos y su distribución cultural.  
- Generar insights para comprender la evolución histórica del yacimiento.

## 🛠️ Tecnologías
- Power Query  
- DAX  
- Power BI  

## 📝 Metodología
1. **Carga de datos:** Importación de archivos CSV (`Registro_de_Campo_raw.csv`, `Analisis_Laboratorio.csv`).  
2. **Limpieza y transformación:** Ajustes de texto, fechas y números; eliminación de inconsistencias.  
3. **Modelado:** Creación de relaciones, tabla calendario y medidas DAX para análisis dinámico.  
4. **Visualización:** Dashboards con KPIs, gráficos de distribución de artefactos, líneas de tiempo y filtros interactivos.  
5. **Validación:** Revisión de resultados y consistencia de datos para asegurar precisión en insights.
